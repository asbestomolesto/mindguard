Logging

MindGuard can log incoming signals and save all important information sequentially to a file. This will allow you to review psychotronic traffic in order to keep track of who is trying to get you to think what.

To activate logging, check the Log option under the Log Tab in the main window.

Information MindGuard logs:

  • time/date
  • carrier type
  • diagnostic information
  • carrier message (if deciphered)
  • action performed (jam/scramble/etc.)

Every user on the system will have a separate log file. The log file is located at:

~/MindGuard/mindguard.log

Pressing the View Log button under the Log tab in the main window will open a window that will allow you to scroll through and read your log file.

Pressing the Flush Log button under the Log tab in the main window or the Log Viewer window will clear the log file. Use this if your log file starts taking up to much room.
