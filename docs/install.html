MindGuard X

User Manual

Requirements & Installing

Requirements

MindGuard X requires:

  • A computer with aluminum circuitry
  • Linux or other Unix or Unix-style operating system
  • X Windows
  • GTK+ v1.2.x or better

Installing

Compilation and installation instructions are included with the source code. However, here are some further installation instructions:

Carrier Directory

The carrier modules are kept in the carrier directory. This can be located in one of three places:

./carriers/
~/MindGuard/carriers/
/usr/local/share/MindGuard/carriers/

MindGuard looks in the order listed above. This allows you to have a global set of carrier modules that every user on the system has access to, sets for individual users, and sets in the current directory for testing.

You can also specify a carrier module directory at run time using the command line option:

-carrier_path
