Diagnostic Display

MindGuard's Diagnostic Display gives textual information on incoming psychotronic signals from both the scanner and the jammer. It is located on the right half of the main window under the top row of buttons.

Scanning/Deciphering Messages:

  • "Signal Isolated"
    Signal was isolated and will be sent to be deciphered.

  • "Signal Not Isolated"
    Unable to isolate signal to be deciphered.

  • "Carrier: xxx"
    Gives the carrier of the isolated signal.

  • "Context: xxx"
    What type of information is contained in the isolated signal. This can include the following:

    • Linguistic
      Acting on the higher reasoning center of the brain. Textual description is given.

    • Visual
      Psycho-symbolic imagery and visual sensations. Textual description given.

    • Sensual
      Nonvisual sensations including, but not limited to, touch, taste, smell, and sound. Textual description given.

    • Physio-Emotive
      Psychological and physiological states. Textual description given.

  • "Source: xxx"
    Most likely source of the signal (moving vector, etc.)

  • "Info: xxx"
    Gives various diagnostic data (for advanced users only.)

  • "Undecipherable"
    MindGuard couldn't decipher an isolated signal.

Jamming Messages:

  • "Carrier Hit Detected"
    MindGuard detected a carrier signal.

  • "Carrier: xxx"
    Gives carrier type detected.

  • "Jamming..."
    MindGuard is attempting to jam the signal.

  • "Scrambling..."
    MindGuard will scramble the carrier.

  • "SUCCESSFUL"
    Signal was negated or made inert.

  • "FAILURE"
    MindGuard was unable to defeat carrier. Please see What To Do In Case Of A Failure if this should occur.

Carriers:

If you select a carrier module from the list, relevant information such as its name, version, and copyright status will be displayed in the diagnostic box. When reloading the carrier modules, the carrier module directory and number of modules will be displayed.
